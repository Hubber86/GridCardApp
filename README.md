# GridCardApp

A responsive Testimonials page layout built using HTML5 and CSS3.

## 📌 Project Overview

This project focuses on building a modern testimonial card layout using:

- CSS Grid
- Grid Template Areas
- Grid Columns
- Flexbox alignment
- Responsive design using media queries

## 🎯 Key Concepts Practiced

- Grid Template
- Grid Area
- Grid Column
- Responsive breakpoints
- Flexbox alignment
- CSS variables
- Clean UI structure
